## 1. 品牌基础

- 品牌名：Donut（正式主品牌统一使用 Donut，"Donut AI" 只在账号名、SEO、解释性语境中辅助出现，不作为主品牌名单独使用）
- 一句话：Agentic Trading for Every Market.
- 副标题：你的私人 AI，全天候监控、分析、推荐并执行——覆盖所有金融市场。
- 主域名：getdonut.ai
- 默认 CTA：Launch Donut

适用范围：官网、社媒、PR、Deck、BD、社区、招聘、对外介绍等所有对外沟通材料。

## 2. 产品定义与范式

Donut 不是：
- 一个「资讯工具」
- 一个「单纯聊天机器人」
- 一个「只服务 crypto 的 browser 叙事」

Donut 是：
- Agentic Trading 产品。
- 本质：让 AI 持续监控市场、主动发现机会、给出判断，并沿最优路径执行。

今天的入口：D0 Agent  
明天的终局：Zero-human funds 的市场基础设施（Donut 作为 autonomous capital 的执行基础设施）。

关键信息：Donut = 从「帮你看」走向「帮你做」的交易基础设施。

## 3. 问题空间与用户痛点

传统交易的问题不是「没有信息」，而是「信息太多、反应太慢、执行太碎」。

用户痛点：
- 看不过来：市场 24/7，机会出现得比人快。
- 看到了也做不动：跨场所、跨链、跨资产执行太复杂。
- 做了也未必最优：路径、价格、费率、时机都在损耗收益。

Donut 的答案（产品三段式）：
- 监控：全天候盯市场。
- 判断：主动分析和推荐。
- 执行：寻找最优路径完成交易。

## 4. 核心定位

- 核心品类：Agentic Trading
- 核心差异：Proactive，不是 Reactive

Reactive 产品：
- 你问，它答
- 你点，它执行
- 你不动，它也不动

Donut：
- 主动监控
- 主动发现
- 主动提醒
- 主动建议
- 在授权边界内主动执行

定位总结：这不是功能升级，而是产品范式变化（从「被动工具」到「主动 Agent」）。

## 5. 产品矩阵与品牌结构

产品矩阵：
- D0 Agent — 你的私人 AI 交易员
  - 全天候监控市场、发现机会、自动执行——你不用盯盘。
- Trading Platform — 为 Agent 而生
  - 跨场所路由、最优执行、行业最低 Agent 费率。
- Assets — 超越加密货币
  - 交易永续合约、链上股票、大宗商品——所有资产，一个平台。

品牌结构：
- 主品牌：Donut — 母品牌
- 产品入口：D0 Agent — 面向用户的 Agent
- 能力底座：Trading Platform — 执行基础设施
- 扩张方向：Assets — 多资产覆盖

## 6. 双叙事框架（2C / VC）

2C — 面向用户：
- 大主线：看到别人看不到的。
- 核心不是「我们有多少技术」，而是「你会比别人更早看见机会」。
- 关键承诺：
  - 看到别人看不到的全市场动态。
  - D0 全天候监控所有市场，在机会变得显而易见之前就浮现出来。
  - 一个 Agent，覆盖所有市场，最优路径，永不停歇。
- 用户真正买单的：
  - 更早发现
  - 更低磨损
  - 更少盯盘
  - 更强执行

VC — 面向投资人：
- 总句：Zero-Human Funds 的基础设施。
- 路线：
  - 今天先用零售交易员验证需求。
  - 明天让每个 AI fund 跑在 Donut 上。
- VC 视角关键词：
  - execution layer
  - cross-venue routing
  - best execution
  - agent-native platform
  - infrastructure for autonomous capital
- 注意：DEX 是基础设施叙事的一部分，不单独当作面向用户的卖点推。

## 7. 目标用户

阶段一用户画像：
- 1–2 年以上经验的交易者
- 高频关注市场
- 有一定资金量
- 对速度、执行、机会敏感
- 被市场教育过，不再相信空话

他们不需要：
- Crypto 101
- 概念科普
- 空泛口号

他们需要：
- 真机会
- 真效率
- 真执行
- 真低损耗

## 8. 品牌语言与语感

我们怎么说话：
- 专业
- 克制
- 信息密度高
- 不油腻、不喊口号

我们的语感：
- sharp
- direct
- first-principles
- 有判断，不浮夸。

避免的表达：
- revolutionary
- game-changer
- guaranteed profits
- 稳赚 / 轻松暴富
- 任何收益承诺

## 9. 对外表达红线

禁止/谨慎区：
- 不再使用 Donut Browser（不再把 Browser 当成主叙事）。
- 不再把 HIL / Auto Research / Perps 五条线平铺出去。
- 不承诺收益、不暗示保本。
- 不夸大当前尚未 fully launched 的能力。
- 不把基础设施能力讲成「用户当下已经完整可用的一切」。
- 可以讲「方向」，但必须清晰区分「现有能力」 vs 「未来愿景」。

## 10. 核心卖点（对外可直接引用）

1) Proactive Intelligence  
- 主动发现，不等你问。

2) Best Execution  
- 不是只帮你下单，而是帮你找最优路径（跨场所、跨资产、考虑价格/费率/滑点/时机）。

3) All-Market Ambition  
- 不止 crypto，长期覆盖所有金融市场。

4) Agent-Native Platform  
- 不是把 AI 硬塞进旧交易界面，而是从底层按 Agent 的工作方式重构平台。
